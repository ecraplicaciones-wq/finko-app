# Finko Pro 🇨🇴💚

> App de finanzas personales para Colombia. Funciona offline como PWA instalable.

---

## Tecnología

| Capa | Tecnología |
|---|---|
| Frontend | Vanilla JavaScript (ES Modules) |
| Persistencia | `localStorage` (objeto `S`) |
| Offline | Service Worker (Cache First + SWR) |
| Estilos | CSS puro con variables (tema claro/oscuro) |
| Exportación | SheetJS (lazy-load) |
| Fuentes | Inter + DM Mono (Google Fonts) |
